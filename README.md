# 🔍 Smart Port Scanner

A GUI-based Network Port Scanner developed using Python.  
This tool allows users to scan open and closed ports on a target system efficiently.

---

## 🚀 Features
- Scan both open and closed ports
- User-friendly GUI interface
- Multi-threaded fast scanning
- Service detection (HTTP, SSH, etc.)
- Save scan results to a file
- Supports IP and hostname

---

## 🛠️ Technologies Used
- Python 3
- Socket Programming
- Threading
- Tkinter (GUI)

---

## ▶️ How to Run
1. Install Python
2. Download or clone this repository
3. Run:

```bash
python port_scanner.py
